# 2026-06-02 工作日志

## 喵喵猜词 — 修复首轮说书人倒计时不生效

**Bug**：开局第一轮说书人（storyteller_picking 阶段）的倒计时 UI 不显示/不倒数，但后续轮次正常。

**根因**（`games/catguess/src/components/GameScreen.vue:836`）：
GameScreen 组件通过 `App.vue` 的 `v-else-if="gameState.screen === 'game'"` 条件渲染。
游戏开始时，`handleStartGame()` 先设置 phase → `updateLocalState` → `syncScreenToPhase` 将 screen 切为 `game`，然后 GameScreen **首次挂载**。
此时 `watch(phase, ...)` 创建时 phase 已经是 `storyteller_picking`，未发生"变化"，watch 回调不触发，`startCountdown()` 从未被调用。
后续轮次正常是因为组件已挂载，watch 能检测到 phase 过渡。

**修复**：给 `watch(phase, ...)` 加 `{ immediate: true }`，组件挂载时立即执行一次回调，检查当前 phase 是否为计时阶段并启动倒计时。
